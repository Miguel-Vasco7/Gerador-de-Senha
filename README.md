## 📦 Instalação

Para usar o Gerador de Senhas, siga estas etapas simples:

**Clone o repositório**:
   ```bash
   git clone https://github.com/Miguel-Vasco7/Gerador-de-Senha.git
   ```

2. **Entre no diretório**:
   ```bash
   cd Gerador-de-Senha
   ```

3. **Execute o programa**:
   ```bash
   python Gerador-de-Senha.py
   ```
